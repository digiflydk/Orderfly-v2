# Orderfly developer overview

Orderfly is a multi brand ordering platform for restaurants.

This document gives a high level overview of the system:

- Next.js frontend and superadmin UI
- Firebase / Firestore backend
- Stripe payments (if relevant)
- Customer webshop flows (browse menu, add to cart, pay)
- Restaurant admin flows (order management, basic reporting)

Use the other docs under `/superadmin/docs` for details on API, database structure and auditing.
